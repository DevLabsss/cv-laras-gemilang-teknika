# CV. Laras Gemilang Teknika

Website resmi perusahaan CV. Laras Gemilang Teknika.

## Tentang Perusahaan

CV. Laras Gemilang Teknika adalah perusahaan yang bergerak di bidang jasa teknik dan konstruksi, dengan fokus pada pelayanan profesional, kualitas pekerjaan, serta kepuasan pelanggan.

## Fitur Website

- Landing page modern dan responsif
- Profil perusahaan
- Informasi layanan
- Statistik perusahaan
- FAQ (Frequently Asked Questions)
- Informasi kontak
- Optimasi untuk desktop, tablet, dan mobile

## Teknologi yang Digunakan

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Struktur Project

```text
Project/
├── assets/
├── index.html
├── style.css
├── script.js
└── README.md
```

## Menjalankan Project

Clone repository:

```bash
git clone https://github.com/DevLabsss/CV.-Laras-Gemilang-Teknika.git
```

Masuk ke folder project:

```bash
cd CV.-Laras-Gemilang-Teknika
```

Buka file `index.html` menggunakan browser.

## Deployment

Website di-host menggunakan GitHub Pages dan dapat diakses melalui:

https://larasgemilangteknika.com/
