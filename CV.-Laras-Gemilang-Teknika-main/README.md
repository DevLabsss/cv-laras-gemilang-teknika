# CV. Laras Gemilang Teknika

Website resmi perusahaan **CV. Laras Gemilang Teknika** yang bergerak di bidang penyediaan material industri, konstruksi, dan kebutuhan teknik.

## 🌐 Live Website

https://larasgemilangteknika.com/

---

## 📌 Features

* Responsive Modern Landing Page
* Product Catalog Section
* Company Profile
* Smooth Animation
* Contact Information
* Mobile Friendly Design

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* Tailwind CSS
* Font Awesome

---

## 📂 Project Structure

```bash
assets/        # Images, icons, logos
css/           # Stylesheets
js/            # JavaScript files
index.html     # Main page
```

---

## 🚀 Installation

Clone repository:

```bash
git clone https://github.com/DevLabsss/CV.-Laras-Gemilang-Teknika.git
```

Open project:

```bash
cd CV.-Laras-Gemilang-Teknika
```

Run locally using Live Server or open `index.html`.
